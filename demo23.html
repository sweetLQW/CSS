position属性

参数：absolute、fixed、relative、static、inherit


提示：absolute与fixed定位会脱离文档流，布局方式会更改为inline-block。


absolute 绝对定位

脱离文档流，不占位，相对于 static 定位以外的第一个父元素进行定位，如果没有则根据HTML文档进行定位（不是body与html元素），会随着滚动条滚动，元素的位置可通过 "left", "top", "right" 以及 "bottom" 属性进行规定。

我是绝对定位元素（left:0,top:20px）

fixed 固定定位

脱离文档流，不占位，根据浏览器的窗口进行定位，不随滚动条滚动，元素的位置可通过 "left", "top", "right" 以及 "bottom" 属性进行规定。

我是固定定位元素（left:0,top:0）

relative 相对定位

不脱离文档流，根据元素自身的位置进行定位，元素的位置可通过 "left", "top", "right" 以及 "bottom" 属性进行规定。


static 默认值

没有定位，不能使用left等定位及z-index。


inherit 继承

从父元素继承 position 属性的值。
